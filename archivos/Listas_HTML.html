Actividad 1

HTTP
Es un protocolo de la capa de aplicación para la transmisión de documentos hipermedia, como HTML.
HTML
siglas en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web.
URL
Dirección web, en referencia a un recurso web que especifica su ubicación en una red informática y un mecanismo para la recuperación de esta misma.
TCP/IP
Protocolo de enlace de datos que se usa en Internet para que los ordenadores y otros dispositivos envíen y reciban datos.
Internet
Red informática de nivel mundial que utiliza la línea telefónica para transmitir la información.
W3C
World Wide Web Consortium, es un consorcio internacional que genera recomendaciones y estándares que aseguran el crecimiento de la World Wide Web a largo plazo.
Mozilla Foundation
Es una organización sin ánimo de lucro dedicada a la creación de software libre. Tiene como misión «mantener la elección y la innovación en Internet».
